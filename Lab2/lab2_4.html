Добавление встроенных фреймов

Один из способов добавления содержимого на страницу — это внедрить другую HTML-страницу в текущую. Это делается с помощью встроенного фрейма или элемента <span class="tag"><iframe></span>. Данный элемент принимает URL другой HTML-страницы в значении атрибута <span class="attribute">src</span> — это вызывает содержимое из внедрённой HTML-страницы для отображения на текущей странице. Значение атрибута <span class="attribute">src</span> может быть URL относительно страницы, где появляется элемент <span class="tag"><iframe></span> или абсолютным адресом для исключительно внешней страницы.

Многие страницы используют элемент <span class="tag"><iframe></span> для внедрения медиа-контента на страницу с внешнего сайта, вроде Google Maps, YouTube и др.
